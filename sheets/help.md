Cheatly Help
============

Usage
-----

To list the available

    cheatly list

To see the sheet

    cheatly show $SHEET_NAME

To create a new sheet local:

1. `git clone https://github.com/arthurnn/cheatly.git`
2. `cheatly new $SHEET_NAME`
3. Create a pull request

To see this message

    cheatly help

happy cheating
